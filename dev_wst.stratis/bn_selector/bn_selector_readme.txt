Как пользоваться:
Настраивается несколькими последовательно выполняемыми функциями.
Обратите внимание: скрипт работает локально на той машине, на которой запущен. Поэтому:
- нужно следить за локальностью выполняемых команд, при необходимости использовать remoteExec;
- нужно определить, на какой машине выполнять скрипт (например, назвать одного из юнитов ks_red и выполнить следующую проверку: 

{waitUntil player == player};

        if (player == ks_red) then {

                // описанные ниже команды

        }

- некоторые глобальные команды (тот же setPos) начинают нормально транслироваться только после брифинга. Нужно тестировать миссию на выделенном сервере.
- чтобы интерфейс отрисовался на брифинге, нужно выполнять описанные функции до любых sleep'ов и т.д.

Исполняемый код получает локальную булеву переменную _bn_default, которая равна false, если выбор в данной группе сделан командиром, или true, если выбор командиром не сделан и скрипт использовал дефолтный вариант.

1. Инициализация: положить в папку с миссией и выполнить на нужной машине (например, у командира стороны) скриптовый файл bn_selector.sqf:

[] call compile preprocessFileLineNumbers "bn_selector.sqf";


Скрипт инициализирует необходимые функции.

2. Добавить группу переключателей функцией bn_selector_create_group (параметры: 0 - название, String):

["Средство усиления:"] call bn_selector_create_group;

Необязательные параметры функции bn_selector_create_group: 
максимальное количество одновременно выбранных опций, по умолчанию 1;
возможность не выбирать ни одного из вариантов (по умолчанию false, т.е. при пустом выборе будет принудительно выбран дефолтный вариант)

Например: ["Средство усиления:",3,true] call bn_selector_create_group;
позволит выбрать от 0 до 3 средств усиления из списка.


3. Добавить несколько вариантов на выбор в данной группе функцией bn_selector_add_item (параметры: 0 - название, String; 1 - выполняемый код, String; 2 (необязательно) - координаты на карте, Array).

["ЗУ на севере","my_zu_position = [1800,6000]",[1800,6000]] call bn_selector_add_item;
["ЗУ на юге","my_zu_position = [1600,5300]",[1600,5300]] call bn_selector_add_item;



Повторять пункты 2-3 до полного удовлетворения.

4. Завершить отрисовку меню командой bn_selector_paint (параметры: 0 - координаты меню на карте, Array; (далее необязательно) 1 - шаг между пунктами меню по вертикали (на картах с неправильной системой координат поставить положительный), Number; 2 - код, выполняемый после завершения выбора, String). 


[] call compile preprocessFileLineNumbers "bn_selector.sqf";   

["Средство усиления:"] call bn_selector_create_group;

        ["Танк Т-72Б","'rhs_t72bc_tv' createVehicle (position player)"] call bn_selector_add_item;

        ["Вертолёт Ми-24","'RHS_Mi24V_vvs' createVehicle (position player)"] call bn_selector_add_item;

        ["Миномётный взвод","'BN_2b14_box' createVehicle (position player)"] call bn_selector_add_item;

["Выбери положение ЗУ-23-2 на карте."] call bn_selector_create_group;

        ["ЗУ на севере","my_zu_position = [1800,6000]",[1800,6000]] call bn_selector_add_item;

        ["ЗУ на юге","my_zu_position = [1600,5300]",[1600,5300]] call bn_selector_add_item;

[[1000,6000]] call bn_selector_paint;

 

sleep 1; // это произойдёт уже после брифинга

zu setPos my_zu_position;
